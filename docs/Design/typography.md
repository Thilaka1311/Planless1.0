# 13. Typography

# Planless Typography System

Version: 1.0

This document defines the complete typography system used throughout Planless.

Typography never changes between categories.

Sports, Movies, Restaurants, and Core Planless all share the exact same typography.

Only color changes.

The goal is clarity, rhythm, hierarchy, and consistency.

---

# Typography Philosophy

Typography should feel invisible.

The interface should never feel "designed around fonts."

Users should effortlessly understand

- what is important
- what is secondary
- what is actionable

Hierarchy should come from

- weight
- spacing
- size

—not excessive colors.

Typography should feel calm, modern, premium and highly readable.

# Typography Rules

Never use more than three font sizes inside one component.

Never mix multiple bold weights together.

Prefer whitespace over larger typography.

Body text should remain Regular (400).

Titles should remain Semibold (600).

Display text should remain Bold (700).

Avoid italic text.

Avoid underlined text.

Never use ALL CAPS for headings.

Only tiny badges may use uppercase.

Never reduce body text below 14px.

Never use decorative fonts.

Never adjust tracking manually except for display typography.

Typography should remain identical across every category.

Only colors change.

Never let typography compete with photography or illustrations.

Content should always remain the hero.
